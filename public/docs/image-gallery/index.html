使用方法

要在您的 Markdown 内容中使用可折叠图片，请使用以下语法：

{{< foldable-image 
    src="图片URL" 
    alt="图片描述" 
    caption="图片说明文字" 
    color="颜色名称" 
    width="图片宽度" 
    open="true或false" 
>}}

参数说明

  • src: 图片的 URL 地址（必填）
  • alt: 图片的替代文本，也用作折叠标题（必填）
  • caption: 图片下方的说明文字（可选）
  • color: 边框颜色，可选值包括 red、blue、green、purple、orange、teal、yellow、pink、gray（默认为 blue）
  • width: 图片宽度，可以是百分比或像素值（默认为 100%）
  • open: 是否默认展开图片，值为 “true” 或 “false”（默认为 “false”）
