数据库基础

数据库是有组织的数据集合，通常以电子形式存储在计算机系统中。

关系型数据库

关系型数据库使用表格来组织数据：

SQL基础

SQL（结构化查询语言）是用于管理关系型数据库的标准语言：

-- 创建表
CREATE TABLE users (
  id INT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100),
  created_at TIMESTAMP
);

-- 插入数据
INSERT INTO users (id, name, email, created_at)
VALUES (1, '张三', 'zhangsan@example.com', NOW());

-- 查询数据
SELECT * FROM users WHERE id = 1;

-- 更新数据
UPDATE users SET name = '李四' WHERE id = 1;

-- 删除数据
DELETE FROM users WHERE id = 1;

常见的关系型数据库

  • MySQL
  • PostgreSQL
  • SQLite
  • Oracle
  • SQL Server

NoSQL数据库

NoSQL数据库不使用传统的表格模型：

文档数据库

MongoDB是一种流行的文档数据库：

// 创建文档
db.users.insertOne({
  name: "张三",
  email: "zhangsan@example.com",
  created_at: new Date(),
  tags: ["vip", "active"]
});

// 查询文档
db.users.find({ name: "张三" });

// 更新文档
db.users.updateOne(
  { name: "张三" },
  { $set: { name: "李四" } }
);

// 删除文档
db.users.deleteOne({ name: "李四" });

其他NoSQL数据库类型

  • 键值存储（Redis）
  • 列存储（Cassandra）
  • 图数据库（Neo4j）

数据库设计原则

  • 规范化：减少数据冗余
  • 索引：提高查询性能
  • 事务：确保数据一致性
  • 安全性：保护数据不被未授权访问
