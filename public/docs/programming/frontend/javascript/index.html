JavaScript Basics

Introduction

JavaScript is a programming language that enables interactive web pages. It is an essential part of web applications, and nearly all modern websites use it.

Variables

In JavaScript, you can declare variables using var, let, or const:

// Using var (older way)
var name = "John";

// Using let (modern way, block-scoped)
let age = 30;

// Using const (for values that won't change)
const PI = 3.14159;

Functions

Functions in JavaScript can be defined in several ways:

// Function declaration
function greet(name) {
  return `Hello, ${name}!`;
}

// Function expression
const sayGoodbye = function(name) {
  return `Goodbye, ${name}!`;
};

// Arrow function (ES6+)
const multiply = (a, b) => a * b;

DOM Manipulation

JavaScript can modify HTML elements:

// Change text content
document.getElementById("demo").textContent = "New text";

// Change styles
document.getElementById("demo").style.color = "red";

// Add event listeners
document.getElementById("button").addEventListener("click", function() {
  alert("Button clicked!");
});

Modern JavaScript Features

ES6 and later versions introduced many useful features:

  • Template literals
  • Destructuring
  • Spread/rest operators
  • Classes
  • Modules
  • Promises and async/await

Conclusion

JavaScript is a versatile language that continues to evolve. Learning JavaScript fundamentals is essential for web development.
